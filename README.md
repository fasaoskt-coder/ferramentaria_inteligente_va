# Ferramentaria Elétrica CA — Vale (HTML)

Projeto estático (apenas front-end) para publicar no GitHub e na Vercel.

## Estrutura
- `index.html` (app)
- `manifest.json` + `icon-192.svg` + `icon-512.svg` (PWA básico)
- `vercel.json` (rotas para servir o `index.html`)

## Deploy na Vercel
1. Suba este repositório no GitHub
2. Na Vercel: **New Project** -> selecione o repo
3. Framework Preset: **Other**
4. Build Command: (vazio)
5. Output Directory: `/`
